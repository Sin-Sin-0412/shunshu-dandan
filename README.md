### 🧊 Overview
「春愁だんだん」は、Three.jsによって構築された3Dインタラクティブ・エクスペリエンスです。
春の午後の日差し、そして階段（だんだん）。ただそこにある空間です。

### 🎸 Features
- **Atmospheric 3D Scene**: 15時の日差しを再現したライティングと、FilmGrainによる質感。
- **Music Player**: 空間に溶け込むプレイヤーで、厳選された3曲を聴く。
- **Organic Transition**: SVGフィルター（feTurbulence）を使用した、有機的な溶解演出。文字と背景がノイズの中に溶けていく「侵食」の表現。
- **Adaptive Interaction**: マウスの動きに呼応するUIレイヤーと、呼吸するように微細に揺れるカメラ。

### 🛠 Tech Stack
- **Engine**: Three.js
- **Animation**: GSAP (GreenSock)
- **Asset Loading**: GLTFLoader / LoadingManager (Minimum 2s display guarantee)
- **Post Processing**: UnrealBloom, Bokeh, FilmPass
- **Deployment**: Netlify
